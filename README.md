# ansible_Bash_Playbook

## How To Use

- Create Jenkins A job
- Provide a Job Name jenkins Job Type = Pipeline then OK
- Select as IMAGE Option Below
- REPOSITORY URL - https://github.com/harishchanderdalal/ansible_Bash_Playbook.git
- same and RUN JOB
